<h2 class="code-line" data-line-start=7 data-line-end=8 ><a id="Deskripsi_aplikasi_7"></a>Deskripsi aplikasi</h2>
<p class="has-line-data" data-line-start="9" data-line-end="10">Sitem aplikasi PWA dibuat dengan bertujuan agar memudahkan karyawan dalam ruang lingkup kerja dalam suatu perusahaan. Berikut beberapa deskripsi kegunaan aplikasi PWA ini:</p>
<ol>
<li class="has-line-data" data-line-start="10" data-line-end="11">Mengerjakan tugas laporan dan historynya</li>
<li class="has-line-data" data-line-start="11" data-line-end="12">Berisi Informasi atau Biodata setiap karyawan</li>
<li class="has-line-data" data-line-start="12" data-line-end="13">Digunakan untuk absensi karyawan</li>
<li class="has-line-data" data-line-start="13" data-line-end="14">Monitoring laporan pekerjaan karyawan tersebut dengan atasan maupun antar divisi</li>
<li class="has-line-data" data-line-start="14" data-line-end="15">Memanajemen dokumen surat-surat perusahaan</li>
<li class="has-line-data" data-line-start="15" data-line-end="16">Memanajemen menggunakan metode(create, upload, delete) serta monitoring Aset dan resiko pada perusahaan</li>
<li class="has-line-data" data-line-start="16" data-line-end="17">Mencakup informasi tentang perusahaan dan dokmentasi-dokumentasi yang ada di perusahaan.</li>
<li class="has-line-data" data-line-start="17" data-line-end="18">Terdapat user account, untuk mengontrol akun dari segi pengguna/user (karyawan).</li>
</ol>
